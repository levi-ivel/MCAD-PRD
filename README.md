# MCAD-RPD

## Overzicht

Deze repository bevat de source voor MCAD-RPD, een kleine analyse van de Marine Cyber Attack Database (MCAD) gemaakt in R, DuckDB en Python.

## Installatie

Voer het setup script uit:
```bash
bash setup.sh
```

## Gebruik

Data verwerken:
```bash
Rscript src/main.r
```

Data bekijken:
```bash
python src/display.py
```
